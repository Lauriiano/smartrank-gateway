## Descrição

Gateway para gerenciar requisições de microserviços smartrank. Desenvolvido para estudo.

## Rodando o Gateway

```bash
# Subir container com imagem do rabbitMq no docker
$ docker-compose up -d

# Instalar dependencias
$ npm install

# subir o servidor
$ npm run start:dev
```
